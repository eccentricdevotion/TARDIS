  • create_worlds: true in the config, and a multi-world plugin
  • OR
  • a special world called TARDIS_TimeVortex — to learn how to set this up visit the Time Vortex page

Growing rooms

You can create rooms in your TARDIS. Each room costs a varying amount of Artron Energy. To create a room, simply place the required material (the seed) adjacent to the side of the pressure plate that faces you when looking at the door.

Use the command:

/tardis room [room type]

and follow the instructions. The room types and their corresponding seed blocks are as follows: 

Empty - GLASS
Arboretum - LEAVES
Bedroom - GLOWSTONE
Cross roads - SOUL SAND
Eye of Harmony - BRICK STAIRS
Farm - DIRT
Gravity well - MOSSY COBBLESTONE
Greenhouse - MELON block
Anti-Gravity well - SANDSTONE
Kitchen - PUMPKIN
Library - BOOKCASE
Mushroom - GRAVEL
Passage - CLAY block
Passage (Long) - NOTE block
Pool - SNOW block
Exterior Rendering Room - HARDENED_CLAY block
Secondary console - ENDER STONE/WOOD
Vault - DISPENSER
Workshop - NETHER BRICK block

Notes:

  • The Eye of Harmony room type is harmony
  • The gravity well room type is gravity and requires an empty ARS room slot below the room as well.
  • The anti gravity well room type is antigravity and also requires an additional ARS room slot below.
  • The exterior renderer room type contains no doors for manual entry. Use of the scanner will teleport you into this room if it exists within your tardis.
  • The secondary console room types are: baker and wood.

Once the room is finished generating, a door will appear at the chosen location and a confirmation message will be shown.

Changing the room wall block

By default (most) room walls are ORANGE wool. If you would like to replace all the ORANGE wool blocks with another material, you can set a wall player preference before growing the room.

To see a list of available wall materials, see the walls page, or type:

/tardisprefs wall help

To set your wall block preference, type:

/tardisprefs wall [material]

Where [material] is a block from the wall help list, e.g:

/tardisprefs wall huge_mushroom_2

The next room your grow (if it normally contains ORANGE wool) will have the blocks replaced with your preference.

Changing the floor block

Similar to the wall player preference, there is also a floor player preference. If you would like to replace all the LIGHT GREY wool blocks (the default for room floors) with another material, you can set the floor player preference before growing the room. Use:

/tardisprefs floor [material]

Room growing tips

  1. TARDIS rooms were designed to be modular
  2. Start growing the room from the centre of the wall.
  3. While not essential, rooms work best when connected by passages. This will ensure that rooms don’t overlap, and it is easier to connect them all up nicely.
  4. When clicking the seed block with the TARDIS key make sure you are facing square on to the wall.

Removing rooms

Rooms may also be removed (or in TARDIS terms - jettisoned).

Simply place TNT on the interior entrance of the room and use the command:

/tardis jettison [room type]

Then follow the onscreen instructions.

Note: Always face into the room you want to jettison, otherwise you are likely to be removing parts of your TARDIS that you don’t want to.

Gravity wells

TARDIS gravity wells let you travel in any direction without having to walk. You can grow a gravity well that goes down, and an anti-gravity well that goes up. This allows you to expand the TARDIS interior over as many levels as you desire. Using the /tardisgravity command you can also create ‘sideways’ gravity wells.

Visit the gravity wells page for more specific information.
